# 🎯 Match Pins

A simple mobile-friendly puzzle game where you swap colored pins to match the correct order.

This project was built as a **Progressive Web App (PWA)** using HTML, CSS, and JavaScript and is hosted using **GitHub Pages**.

---

## 📱 Play the Game

👉 Live demo:  
https://YOUR_USERNAME.github.io/match-pins/

---

## 🧠 How the Game Works

- You are given a shuffled set of colored pins
- The goal is to match them back to the correct order
- Tap two pins to swap them
- Try to solve it in the fewest moves possible

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- Service Workers (offline support)
- Web App Manifest (installable app)

---

## 📲 How to Install on iPhone (iOS)

You can install this game like a real app:

### Steps:
1. Open the game link in **Safari**
2. Tap the **Share button**
3. Tap **Add to Home Screen**
4. Name it “Match Pins”
5. Tap **Add**

### Result:
- The game appears on your home screen
- Opens in full-screen (like a native app)
- Can work offline after first load

---

## 🤖 How to Install on Android

Yes — the method is basically the same, but slightly simpler.

### Steps:
1. Open the link in **Chrome**
2. Wait for the install prompt OR tap menu (⋮)
3. Tap **Add to Home screen** or **Install app**
4. Confirm installation

### Result:
- App appears in your app drawer
- Runs in standalone mode like a native app
- Offline support works after first load
